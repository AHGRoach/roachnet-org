Install

Install with SideStore.

Use the unsigned IPA and share it straight into SideStore.

1

Download the unsigned IPA to Files on your iPhone or iPad.

2

Open SideStore or AltStore and add the RoachNet IPA from Files.

3

Let it sign and install, then open RoachNet iOS and pair the Mac lane.

Pairing

Point the phone at your Mac.

The desktop still owns the models, vault, installs, and the RoachTail host lane. The phone stays the carry surface.

Companion URL

Use http://<your-mac-ip>:38111 on a real device.

Pair code

Use the one-time RoachTail join code to mint a peer token for this device.

Fallback

If the bridge goes dark, chat, vault reads, runtime snapshots, and Apps installs stay cached until the desktop comes back.
