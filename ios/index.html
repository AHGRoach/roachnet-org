What it does

The phone lane.

Chat

Pick up RoachClaw sessions without opening the full desktop shell.

Vault

Read RoachBrain memory, indexed files, and archive summaries from the paired Mac.

Runtime

Check models, downloads, and service state, then restart the desktop lane from the phone.

Install

Install with SideStore.

1

Download the unsigned IPA to Files on your device.

2

Open SideStore, go to My Apps, and add the RoachNet IPA from Files.

3

Let SideStore sign and install it with your Apple ID.

Download IPA Open SideStore docs

Pairing

Point the phone at your Mac.

RoachNet iOS needs a companion-enabled RoachNet desktop runtime to pair against.

Companion URL

Use http://<your-mac-ip>:38111 on a real device.

Token

Paste the same companion token your desktop runtime was started with.

Install handoff

Apps installs use the same install-intent flow as apps.roachnet.org and land on the paired desktop.
