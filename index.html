Meet RoachNet

It lives on your machines.

RoachNet is an offline-first suite for AI, archiving, and development. Your models, tools, and data run locally, sync on your terms, and do not vanish when a login expires.

  • Contained install under ~/RoachNet
  • Durable archive for docs, projects, and the web you care about
  • Same reproducible environment across the machines you actually use

RoachClaw

Spin up local AI and keep it close.

Use RoachClaw for local models without pretending every thought needs a remote provider. When the cloud helps, use it. When it does not, the local path is already there.

  • Local models first, not rented tabs by default
  • Runtime you can inspect instead of mystery behavior
  • Phone companion and web access when you want reach
