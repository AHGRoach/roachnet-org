Screens

The iPhone lane.

Four surfaces, plus a real fallback lane.

Chat

Pick up the RoachClaw lane.

Live chat when the Mac is paired, cached RoachBrain replies when it is not.

Apps

Send installs back to your Mac.

Saved apps, recent sends, and queued installs that flush when the desktop lane comes back.

Vault

Read the shelf without opening the shell.

RoachBrain, indexed files, and archive summaries stay close on the phone even when the bridge drops.

Runtime

Check the desktop lane fast.

Models, services, and downloads stay visible, and RoachTail can be armed or disarmed from the phone.

Install

Install with SideStore.

1

Download the unsigned IPA to Files on your iPhone or iPad.

2

Open SideStore or AltStore and add the RoachNet IPA from Files.

3

Let it sign and install, then open RoachNet iOS and pair the Mac lane.

Download IPA Install notes

Pairing

Point the phone at your Mac.

The desktop runtime still owns the models, vault, installs, and the RoachTail host lane. The phone stays the carry surface.

Companion URL

Use http://<your-mac-ip>:38111 on a real device.

Pair code

Use the one-time RoachTail join code to mint a peer token for this device.

Fallback

If the bridge goes dark, chat, vault reads, runtime snapshots, and Apps installs stay cached until the desktop comes back.
