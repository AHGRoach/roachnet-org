Account

Keep the website lane tied to your own devices.

Accounts are for the hosted side of RoachNet: web chat, remembered settings, install history, and the metadata that ties your RoachTail and RoachSync lanes together without exposing local file paths.

What it unlocks

One sign-in across the website lane.

  • Future RoachClaw web chat access
  • Remembered settings and install history
  • RoachTail and RoachSync device metadata tied to one account lane
Privacy

Desktop data stays out of the public page.

This lane is for account state, not for dumping local paths, runtime secrets, or machine-private filesystem details into the site.

Order

Local first. Account second.

Pair the desktop and phone through RoachTail first. Then sign into the website lane when you want the hosted side to remember devices, settings, and future chat history.
