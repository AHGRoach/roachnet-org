The company behind the bunker.

RoachWares LLC exists to build boring, stubborn software for people who would rather own their working set than rent it from a dashboard. RoachNet is the first spine: one native app for AI, archive, games, media, notes, maps, and dev work.

We use online infrastructure when it helps with releases, accounts, support, and public metadata. The product itself still treats the network as optional plumbing. Useful when present. Not load-bearing.

What ships from here.

  • Native RoachNet builds for Apple Silicon Macs.
  • RoachNet iOS companion builds and SideStore feeds.
  • Homebrew casks that install from public release artifacts.
  • Public docs, API surfaces, and release notes written like humans work here.

Operating rule

Keep the public face clean. Keep the user's machine quiet.

The site, GitHub releases, Homebrew tap, SideStore feed, and Convex hooks exist so people can find and install RoachNet. They do not get to become the product. The product is still the native app, the vault, the runtime, and the files sitting where the user can actually reach them.
